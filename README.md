# QuickChange ⚡ - Projet 1 : Interface Utilisateur (UI)

## 📌 Présentation du Projet
Ce projet marque la première étape du développement de **QuickChange**, un convertisseur de devises moderne. L'objectif principal de cette phase était de concevoir et structurer une interface utilisateur (UI) propre, ergonomique et responsive en utilisant uniquement les technologies web fondamentales.

## 🛠️ Technologies Utilisées
* **HTML5** : Structuration sémantique de la carte du convertisseur et des éléments de saisie.
* **CSS3** : Habillage graphique moderne, alignement avec Flexbox, et adaptabilité mobile (Responsive Design).

## 🚀 Fonctionnalités Réalisées
* Design épuré et centré (Card Layout).
* Champs de saisie intuitifs pour les montants et sélections des devises (`EUR`, `USD`, `XOF`).
* Bouton d'action principal et zone d'affichage du résultat prête pour le dynamisme.
